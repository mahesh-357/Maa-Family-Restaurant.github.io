Firebase Setup Guide

Alanati Telugu Kitchen - Backend Configuration

1

Create Firebase Project

  1. Go to Firebase Console
  2. Click "Add Project" and name it "alanati-kitchen"
  3. Disable Google Analytics (optional)
  4. Click "Create Project"
2

Create Web App

  1. In your Firebase project, click the Web icon ()
  2. Register app with nickname "Alanati Website"
  3. Copy the Firebase config object shown
3

Enable Firestore Database

  1. Go to "Build" → "Firestore Database"
  2. Click "Create database"
  3. Choose "Start in test mode" (for development)
  4. Select a location closest to India (asia-south1)

Important: Set proper security rules before going live!

Firestore Security Rules (Production)

Update rules in Firestore → Rules tab:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // Allow anyone to read menu
    match /menu/{document} {
      allow read: if true;
      allow write: if false; // Only admin via console
    }
    
    // Allow anyone to create bookings/contacts
    match /bookings/{document} {
      allow create: if true;
      allow read, update, delete: if false;
    }
    
    match /contacts/{document} {
      allow create: if true;
      allow read, update, delete: if false;
    }
  }
}
